Correlations between record and win-rate in baseball
  • AVG(H): Batting average for hitter
  • R(H): Runs for hitter
  • H: Hits
  • 2B: Doubles (2-base hits)
  • 3B: Triples (3-base hits)
  • HR: Home runs
  • SLG: Slugging percentage
  • OBP: On-base percentage
  • OPS: OBP + SLG
  • RISP: Runner in scoring position
  • ERA: Earned run average
  • SV: Saves
  • IP: Innings pitched
  • BB(P): Base on balls for pitcher
  • R(P): Runs for pitcher
  • WHIP: Walks plus hits divided by innings pitched
  • QS: Quality Starts
  • TBF: Total batters faced
  • AVG(P): Batting average for pitcher
  • PKO: Pick off outs
  • PO: Put outs
